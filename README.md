# Job Board Platform

This project was developed as part of the CodeAlpha Backend Development Internship.

## Description

The Job Board Platform allows users to post and view job opportunities.

Users can:
- Post a new job
- View available jobs
- Store job details in a database

## Technologies Used

- Python
- Flask
- SQLite
- HTML

## Project Structure

CodeAlpha_JobBoard/

├── app.py

├── database.db

└── templates/

    ├── index.html

    └── post_job.html

## Features

- Add job postings
- View all jobs
- Store data using SQLite database
- Simple and user-friendly interface

## How to Run

1. Install Flask

pip install flask

2. Run the application

python app.py

3. Open your browser and visit

http://127.0.0.1:5000
